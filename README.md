# TUN 大一命定人格 v5.1｜GitHub 根目錄版

這一版專門配合目前的 GitHub 上傳方式製作。

## 這版不需要資料夾

所有檔案都放在 GitHub Repository 根目錄，包括：

```text
index.html
app.js
styles.css
asset-01.webp
asset-02.webp
loading-course.webp
result-course.webp
...
```

這是本版本刻意設計的正確結構。

## 上傳步驟

1. 解壓縮 ZIP。
2. 到 GitHub Repository。
3. 選 `Add file → Upload files`。
4. 將解壓縮後的所有檔案一次拖入。
5. 同名檔案直接覆蓋。
6. 按 `Commit changes`。
7. 等待 Cloudflare 自動重新部署。
8. 使用無痕視窗開啟網站，或按 `Ctrl + Shift + R`。
