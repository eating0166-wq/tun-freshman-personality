# TUN 大一命定人格 v4.0｜部署設定

## 一、上線前一定要修改

全專案搜尋：

```text
https://YOUR-DOMAIN.example/
```

替換成正式網址，例如：

```text
https://tun-personality.netlify.app/
```

需要修改的檔案：

- `index.html`
- `assets/config.js`
- `robots.txt`
- `sitemap.xml`

Open Graph 圖片已建立於：

```text
assets/og-cover.jpg
```

## 二、最快部署：Netlify Drop

1. 解壓縮 ZIP。
2. 開啟 Netlify Drop。
3. 把「整個專案資料夾」拖入頁面。
4. 取得網址後，依照上方步驟替換正式網址。
5. 再重新拖曳更新後的資料夾。

## 三、GitHub Pages

將解壓縮後的所有檔案放在 Repository 根目錄，並在：

`Settings → Pages → Deploy from branch → main / root`

啟用部署。

## 四、GA4 與 Microsoft Clarity

開啟：

```text
assets/config.js
```

填入：

```js
ga4MeasurementId: "G-XXXXXXXXXX",
clarityProjectId: "xxxxxxxxxx"
```

留空時不會載入任何追蹤程式。

## 五、社群分享檢查

正式網址部署後，請重新整理 Facebook Sharing Debugger 或 LINE 分享快取，確認：

- 標題
- 描述
- 1200×630 分享圖
- 正式網址

## 六、主要檔案

- `index.html`：頁面結構與 SEO
- `assets/styles.css`：樣式
- `assets/app.js`：測驗邏輯
- `assets/config.js`：網址與分析工具設定
- `assets/analytics.js`：GA4／Clarity 載入
- `manifest.webmanifest`：PWA
- `sw.js`：離線快取
